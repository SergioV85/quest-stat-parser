# Quest Parser App
Node.js serverless app for parsing HTML statistic from Encounter/Quest.ua pages

# Development and deployment
## Prerequisites
- node.js > 10.x
- typescript > 3.5.x
- serverless framework, installed globally

# Development
- run `npm i`
- run `npm run start:local`

# Deploy app
- run `npm run deploy`
